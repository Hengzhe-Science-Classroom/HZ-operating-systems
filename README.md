# Operating Systems · Interactive

Part of [Hengzhe's Science Classroom](https://hengzhe-science-classroom.github.io/hub/) · Tier 4 · Graduate

## What You'll Learn

- Processes and threads
- CPU scheduling
- Synchronization and deadlocks
- Memory management: paging, virtual memory
- File systems
- I/O and device management

- Remzi H. Arpaci-Dusseau & Andrea C. Arpaci-Dusseau, *Operating Systems: Three Easy Pieces* (OSTEP, free)
- Abraham Silberschatz, Peter B. Galvin & Greg Gagne, *Operating System Concepts* (10th ed., Wiley)
- Andrew S. Tanenbaum & Herbert Bos, *Modern Operating Systems* (4th ed., Pearson)
